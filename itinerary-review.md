# 行程評估清單

## 1. 步調檢查
- [ ] 每天景點數量是否合理（建議 3–4 個重點）
- [ ] 景點之間交通時間是否充裕（Google Maps 預估）
- [ ] 是否有預留吃飯、休息的空白時間

## 2. 地理動線
- [ ] 同一天的景點是否集中在同一區域
- [ ] 把每天景點標在地圖上，確認移動路線合理

## 3. 偏好比對
- [ ] 行程符合旅遊風格（文化 / 美食 / 購物 / 自然）
- [ ] 有沒有特別想去但未排到的地方
- [ ] 有沒有興趣不大可替換的行程

## 4. 體力評估
- [ ] 是否有爬山、長距離步行，是否安排足夠恢復時間
- [ ] 最後幾天是否安排較輕鬆的行程

## 5. 實際演練
- [ ] 粗估總費用，確認符合預算
- [ ] 用 Google Street View 預覽景點實際樣貌
- [ ] 確認每個景點開放時間與預訂需求
